# Plan 10.2: Flutter XP Gain Overlay & Level-Up Modal

## Objective
Make earning XP feel great. Replace the plain snackbar with a floating "+XP" animation that rises from the tapped task. Show a full level-up bottom sheet when the user crosses a level boundary. Parse the enhanced claim response from Plan 10.1.

## Context
- `eklavya_mobile/lib/features/dashboard/home_tab.dart` — `_onTaskComplete` (line 59) and task list section (lines 514–561)
- `eklavya_mobile/lib/core/services/dashboard_service.dart` — `completeTask` return type
