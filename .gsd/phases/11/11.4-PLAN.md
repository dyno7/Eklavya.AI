# Phase 11.4: Hybrid JIT API

## Objective
Provide the endpoints the frontend will call to track sessions and evaluate JIT state estimation for the Coach UI.

## Tasks
1. **Coach Status JIT Endpoint (`/api/v1/coach/status`)**
   - Compute `GdiService.calculate_current_gdi(user)` without persisting to the DB instantly, saving overhead but providing true Real-Time accuracy.
   - Formulate returning JSON containing the `GDI` score, the "Commitment" state, and narrative position.

2. **Session Telemetry Endpoint (`/api/v1/metrics/session_start`)**
   - Provide a silent POST endpoint for Flutter to ping upon app foregrounding to populate `UserSessionLog` allowing `V(t)` (opens without tasks) to function.
