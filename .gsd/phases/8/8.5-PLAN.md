# Plan 8.5: Roadmap Goals & Tasks Timeline UI

## Objective
A major requirement for Phase 8: render the generated roadmap in a structured timeline/card format where each item is tappable and completable, giving XP updates and visual feedback.

## Context
- eklavya_mobile/lib/features/goals/goals_tab.dart
- eklavya_mobile/lib/core/services/goals_service.dart
