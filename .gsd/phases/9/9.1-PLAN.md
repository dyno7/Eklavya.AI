    In `repositories.py`, add:
    ```python
    async def get_task_completions_last_n_days(db, user_id: str, days: int = 30) -> list[datetime]:
        """Return updated_at timestamps of all COMPLETED tasks for user in last N days."""
    ```
    Use: `SELECT updated_at FROM tasks WHERE milestone_id IN (SELECT id FROM milestones WHERE goal_id IN (SELECT id FROM goals WHERE user_id=:uid)) AND status='completed' AND updated_at >= NOW() - INTERVAL ':days days'`
    
    Create `eklavya_backend/app/presentation/analytics.py`:
    ```python
    GET /api/v1/analytics/summary → AnalyticsSummary {
      daily_xp: list[int]  # length 7, XP per day (past 7 days, oldest first)
      completion_rate: float  # completed / total tasks (0.0–1.0)
      active_days_last_30: int  # distinct calendar days with ≥1 completion
      total_tasks: int
      completed_tasks: int
    }
    ```
    `daily_xp` = for each of the last 7 days, sum `t.xp_reward` for tasks completed on that calendar day.
    Wire: `app.include_router(analytics_router)` in `main.py`.
