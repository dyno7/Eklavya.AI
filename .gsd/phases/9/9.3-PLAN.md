# Plan 9.3: Behavioral Chatbot — Persistent Chat Memory & Adaptive Motivation

## Objective
Elevate the Guru from a one-shot onboarding tool to a persistent behavioral coach. Persist the last N chat turns in the database so the user can return, modify their roadmap, and receive adaptive follow-up motivation. Integrate coach drift context into chat responses so the Guru proactively addresses drop-off. Final E2E smoke test.
