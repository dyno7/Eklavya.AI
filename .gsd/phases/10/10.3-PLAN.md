# Plan 10.3: Rich Task Cards + Badge Shimmer on Profile

## Objective
Polish the task list on Home tab with richer cards (type icon + time + XP pill + difficulty dot). Add a shimmer glow to earned badges on Profile tab so they pop visually.

## Context
- `eklavya_mobile/lib/features/dashboard/home_tab.dart` — task list section (lines 514–561)
- `eklavya_mobile/lib/features/profile/profile_tab.dart` — `_BadgeCard` widget (lines 265–299)
- `eklavya_mobile/lib/core/theme/app_colors.dart` — color tokens
- `flutter_animate` is already a dependency
