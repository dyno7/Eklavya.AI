# Plan 8.3: Global UI Transitions & Navigation

## Objective
Apply the UX polish requested by the user: horizontal slide transitions globally, and a translucent frosted glass bottom navigation bar.

## Context
- eklavya_mobile/lib/core/router/app_router.dart
- eklavya_mobile/lib/features/shell/main_shell.dart
- eklavya_mobile/lib/core/theme/page_transitions.dart
