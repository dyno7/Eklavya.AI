# Phase 11.3: Narrative Arc & Difficulty Estimator

## Objective
Refactor the Gemini Guru Agent to produce "Movie-style" Narrative Arc Roadmaps and implement the Adaptive Goal Decomposition algorithm. Handle the `google.genai` Hotfix so that roadmap generation repairs itself.

## Tasks
1. **Hotfix: Upgrade to `google.genai`**
   - In `app/agents/guru_agent.py`, replace deprecated `google.generativeai` with the modern `google-genai` SDK implementation constraints. Fix any async wrapper changes necessary for `generate_content`.

2. **Adaptive Difficulty Calculation**
   - Calculate Target User Capability based on completion rates `< 0.8` leading to simplification, and `> 0.8` leading to harder tasks.
   - Inject the calculation (`Base Difficulty * User Capability`) into the Prompt for roadmap generation.

3. **Narrative Arc Scheduler prompt engineering**
   - Update `System_Prompt` inside `GuruAgent`:
     - Sort tasks through sequential phases: `Setup` -> `Rising Action` -> `Climax` -> `Shareability`.
     - Assign low difficulty / low identity shift to Setup.
     - Exponentially grow to high difficulty / high identity shift in Climax.
   - Update `MilestoneItem` / `TaskItem` returning schemas parsing the narrative metadata into the database.
